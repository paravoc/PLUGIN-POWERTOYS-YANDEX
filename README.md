# Browser Run Overlay

Расширение для Яндекс Браузера и Chromium в стиле PowerToys Run. По горячей клавише открывает компактное окно поиска поверх текущей страницы без автоматического открытия новой вкладки.

Текущая схема выдачи:
- слева локальный поиск по вкладкам, закладкам и истории
- справа до 10 тематических страниц из браузера по тому же запросу

## Возможности

- поиск по вкладкам, закладкам и истории
- прямой переход по адресу через `u:` или просто `github.com`
- веб-поиск через URL-шаблон
- клавиатурная навигация `ArrowUp`, `ArrowDown`, `Enter`, `Ctrl+Enter`, `Shift+Enter`, `Esc`
- справа отдельный блок `Страницы по теме` с заголовком страницы и названием сайта
- правый список собирается из локальных данных браузера, без внешнего поискового API
- хранение настроек через `chrome.storage.local`
- Shadow DOM для изоляции интерфейса от стилей сайта

## Установка

1. Откройте страницу расширений:
   `browser://extensions` или `chrome://extensions`
2. Включите режим разработчика.
3. Нажмите `Загрузить распакованное расширение`.
4. Выберите папку проекта.
   Если браузер плохо работает с кириллицей в пути, используйте ASCII-папку, например `C:\Users\smirn\Desktop\plugin-unpacked`.
5. После загрузки откройте страницу `chrome://extensions/shortcuts`.
6. Назначьте или проверьте горячую клавишу `Ctrl+Shift+Space`.

## Быстрая проверка

1. Откройте обычный сайт, например `https://example.com`.
2. Нажмите `Ctrl+Shift+Space`.
3. Введите один из запросов:
   `коты`
   `t: github`
   `b: docs`
   `h: chromium`
   `w: manifest v3`
   `u:example.org`
4. Переключайтесь стрелками по левому списку локальных результатов.
5. Смотрите справа блок `Страницы по теме`:
   там отображаются заголовки и сайты, которые уже встречались в браузере по похожему запросу.
6. Клик по элементу справа выбирает его для предпросмотра и действий `Открыть` / `Новая вкладка` / `Копировать ссылку`.

## Архитектура

- `background/`
  Поисковые провайдеры, работа с настройками и service worker-логика.
- `content/`
  Overlay UI, локальный список результатов, правый тематический список и clipboard helper.
- `shared/`
  Общие константы и утилиты для background, content и options.
- `tools/`
  Вспомогательные скрипты, например генерация иконок.

## Ограничения MVP

- На системных страницах браузера (`chrome://`, `browser://`, магазин расширений и аналогичные страницы) content script не внедряется.
- Веб-поиск пока работает как pseudo-result через URL-шаблон, без отдельного поискового API.
- Горячая клавиша меняется только через интерфейс браузера.
